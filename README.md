# Zepto_SQL_project
# Zepto Inventory & Product Analysis using MySQL

## Project Overview

This project analyzes Zepto's product inventory dataset using MySQL. The objective is to perform data exploration, data cleaning, and business analysis to derive actionable insights regarding pricing, discounts, stock availability, and category performance.

---

## Dataset Information

The dataset contains approximately 3,700 product records with the following attributes:

* Product Name
* Category
* MRP
* Discount Percentage
* Discounted Selling Price
* Available Quantity
* Product Weight
* Stock Status

---

## Project Workflow

### 1. Data Exploration

* Count total records
* Inspect dataset structure
* Identify unique product categories
* Analyze stock availability
* Check category distribution

### 2. Data Cleaning

* Detect NULL values
* Identify invalid prices
* Remove products with MRP = 0
* Verify data consistency

### 3. Business Analysis

#### Top 10 Products with Highest Discounts

Identified products offering the best discount percentages.

#### High MRP Products Out of Stock

Detected premium products currently unavailable.

#### Category Revenue Analysis

Estimated potential revenue generated by each category.

#### Premium Products with Low Discounts

Found products with MRP greater than ₹500 and discount less than 10%.

#### Categories with Highest Average Discounts

Ranked categories based on average discount percentage.

#### Price per Gram Analysis

Calculated value-for-money products using price per gram.

#### Product Size Segmentation

Classified products into:

* Low Weight
* Medium Weight
* Bulk Products

#### Inventory Weight Analysis

Calculated total inventory weight available per category.
